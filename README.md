# 📊 Sales Analysis Dashboard - Power BI

## Overview
This project is a Sales Analysis Dashboard built using Microsoft Power BI. It provides interactive visualizations to analyze sales performance across different business dimensions such as segment, region, sub-category, and order date.

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Discount KPI
- Sales by Segment
- Sales by Region
- Sales by Sub-Category
- Sales Trend by Order Date
- State-wise Sales Distribution

## Tools Used
- Microsoft Power BI Desktop
- Power Query
- DAX
- Excel Dataset

- Sales trend shows consistent growth over time.
